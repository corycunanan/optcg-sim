## OP01-004 — Usopp

**Q:** If this Character leaves the battle area due to an Event used by my opponent, can I draw a card according to this Character's effect?

**A:** No, you cannot.

**Q:** If my opponent activates an Event's [Trigger], can I draw a card according to this Character's effect?

**A:** No, you cannot. Activating a [Trigger] is not the same as activating an Event.

**Q:** If my opponent activates an Event while this Character has a DON!! card given to it, can I choose not to draw a card?

**A:** No, you cannot.

**Q:** At what time does the effect which states "when your opponent activates an Event" activate?

**A:** After the Event has been activated and its effect has been resolved.
