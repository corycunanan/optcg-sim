# Starter Decks ST-01 through ST-04


## ST01-001 — Monkey.D.Luffy

**Q:** Can I use this [Activate: Main] effect to give a Character a DON!! card that has already been given to another Character?

**A:** No, you cannot. A DON!! card that has already been given to a Character is not a "rested DON!! card".

**Q:** When I use this [Activate: Main] effect to give a Character a rested DON!! card, do I need to leave the DON!! card sideways?

**A:** No, the orientation of a DON!! card given to a Character does not matter.


## ST01-002 — Usopp

**Q:** If the power of a Character blocking this Character becomes 6000 or more due to a Counter, does the target of the attack revert back to the original target?

**A:** No. Since the [Blocker] has already been activated, the target of the attack will remain the Character that activated the [Blocker].


## ST01-004 — Sanji

**Q:** Can this Character attack if it has been rested by another effect on the turn in which it is played?

**A:** No, a rested Character cannot attack.


## ST01-005 — Jinbe

**Q:** If I have 2 of these cards on my field, can I use 1 Jinbe's [When Attacking] effect on the other Jinbe?

**A:** Yes, you can. "Other than this card" refers to the Character itself, not the card name.


## ST01-007 — Nami

**Q:** Can I use this card's [Activate: Main] effect to give this card itself a DON!! card?

**A:** Yes, you can.


## ST01-011 — Brook

**Q:** Can I use this [On Play] effect to give only 1 rested DON!! card?

**A:** Yes, you can. When a card's text states "up to" a specified number of cards, you can choose 0 cards.

**Q:** Can I use this [On Play] effect to give 1 DON!! card each to my Leader and 1 Character, or 2 different Characters?

**A:** No, you cannot.


## ST01-012 — Monkey.D.Luffy

**Q:** Can my opponent activate [Blocker] if another Character attacks after this Character has attacked?

**A:** Yes, your opponent can activate [Blocker] because it is not during the battle in which this Character's [When Attacking] effect was activated.


## ST01-013 — Roronoa Zoro

**Q:** Does this [DON!! x1] effect mean this Character gains +1000 power even during my opponent's turn?

**A:** Yes, this Character gains +1000 power during your opponent's turn. This effect is effective during both players' turns.


## ST01-015 — Gum-Gum Jet Pistol

**Q:** If the power of a Character with an original power of 6000 or less becomes 7000 or more due to an effect, can it be K.O.'d by this [Main] effect?

**A:** No, it cannot. This effect can only K.O. a Character with a current power of 6000 or less.


## ST01-016 — Diable Jambe

**Q:** Can my opponent activate [Blocker] if another Character attacks after the selected Character has attacked?

**A:** Yes, your opponent can activate [Blocker] because it is not during the battle in which the Character you selected with this Event card is attacking.

**Q:** Can this [Trigger] effect be used to K.O. Characters like "ST01-012 Monkey.D.Luffy" that do not have their own [Blocker] effect but have the word "[Blocker]" in their effect?

**A:** No, it cannot. Only Characters with their own [Blocker] effect at the time of using this card can be K.O.'d.


## ST01-017 — Thousand Sunny

**Q:** Can I activate this Stage's "You may rest this Stage:" effect on the turn it is played?

**A:** Yes, you can.


## ST02-001 — Eustass"Captain"Kid

**Q:** If my Leader has already attacked during this turn and I use this effect to set it as active, can my Leader attack again?

**A:** Yes, your Leader can attack again.


## ST02-003 — Urouge

**Q:** What happens to this Character's power if the number of Characters I have changes while this Character is battling?

**A:** If you have 3 or more Characters at the Damage Step of the battle, the result of the battle is determined on the basis that this Character gains +2000 power.

**Q:** Is this Character included in the count to determine "If you have 3 or more Characters"?

**A:** Yes, this Character gains +2000 power if you have 3 or more Characters including this Character.

**Q:** Does this [DON!! x1] effect mean this Character gains +2000 power even during my opponent's turn?

**A:** Yes, this Character gains +2000 power during your opponent's turn. This effect is effective during both players' turns.


## ST02-007 — Jewelry Bonney

**Q:** Can I use this card’s effect to add non-green {Supernovas} type cards to my hand?

**A:** Yes, you can.

**Q:** What happens if there aren't any {Supernovas} type cards?

**A:** You place all the cards you looked at from the top of your deck at the bottom of your deck in any order.

**Q:** Can I activate this Character's "You may rest this card" effect on the turn it is played?

**A:** Yes, you can.


## ST02-008 — Scratchmen Apoo

**Q:** Can I rest a DON!! card that is already rested?

**A:** No, you cannot. You must choose up to 1 active DON!! card from your opponent's cost area.

**Q:** Can I rest a DON!! card that has been given to an opponent's Character?

**A:** No, you cannot. You must choose up to 1 active DON!! card from your opponent's cost area.


## ST02-009 — Trafalgar Law

**Q:** If my Character has already attacked during this turn and I use this effect to set it as active, can my Character attack again?

**A:** Yes, your Character can attack again.

**Q:** Can I use this [On Play] effect to set my Leader card as active?

**A:** No, you cannot. "1 of your ... Characters" refers only to Character cards.


## ST02-010 — Basil Hawkins

**Q:** Can the Character attack again if this effect sets it as active?

**A:** Yes, the Character can attack again.

**Q:** If this Character battles my opponent's Character, at what point during the Battle Phase does this Character become active?

**A:** This Character becomes active at the end of the battle if the target of the attack was a Character.

**Q:** Does this Character become active if it attacks my opponent's Leader and my opponent activates [Blocker]?

**A:** Yes, it does.

**Q:** I declared my attack while my opponent had 5 Characters on their field, and my opponent activated the [Blocker] of "OP01-014 Jinbe".
At this time, my opponent chose to play a red Character with a cost of 2 or less from their hand according to the [On Block] effect of "OP01-014 Jinbe", and trashed the Jinbe whose [Blocker] they had activated according to the rules.
In this case, will my Character become active?

**A:** No, your Character will not become active.
Since the target for the attack no longer exists, the Counter Step and Damage Step will be skipped, and the battle will end. In this case, the requirement that "this Character battles your opponent's Character" is not resolved, so your Character cannot be set to active.

**Q:** My opponent's Character battling this Character left the field during the Attack Step, Block Step, or Counter Step due to an effect. In this case, will this Character's effect activate?

**A:** No, it will not.
This effect is activated at the end of the battle upon resolving the power comparison in the Damage Step.
If the Character being battled with leaves the field in the middle of the battle (i.e., during the Attack Step, Block Step, or Counter Step), the battle is suspended at that point.
In such a case, this Character's effect will not be activated because the Damage Step has not been resolved.


## ST02-013 — Eustass"Captain"Kid

**Q:** Can I choose not to set this Character as active at the end of my turn if it is rested with 1 or more DON!! cards given to it?

**A:** No, you must set it as active whenever possible.

**Q:** If this Character is active, can I use this [End of Your Turn] effect to set this Character as active again?

**A:** No. Nothing happens if this Character is already active.


## ST02-014 — X.Drake

**Q:** Do Characters with both {Supernovas} and {Navy} types gain +2000 power?

**A:** No, they do not. Each Character with the {Supernovas} type, {Navy} type, or both types gains +1000 power.

## ST02-016 — Repel

## ST02-017 — Straw Sword

**A:** Yes, you can. In that case, nothing happens for this [Main] effect.

**Q:** Can I use a {Supernovas} type Event card with this [Trigger] effect?

**A:** No, you cannot. You can only play a Character card or Stage card.

## ST03-005 — Dracule Mihawk

**Q:** If I have 1 or less cards in my deck, can I use this Character to attack if it has a DON!! card given to it?

**A:** Yes, you can. However, since the [When Attacking] effect is always activated and requires a player to draw 2 cards whenever possible, if the deck is reduced to 0 cards as a result, the player with 0 cards in their deck will lose the game.

## ST03-010 — Bartholomew Kuma
