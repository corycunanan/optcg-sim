# M5.2 — shadcn Component Library

> Replace hand-built Radix wrappers with shadcn/ui equivalents, add new components for page redesigns, and create the motion.dev utility integration.

---

## Goal

Swap the 7 hand-built `src/components/ui/` components for shadcn/ui equivalents while preserving the existing API surface where possible. Add new shadcn components needed for M5.3 page redesigns. Components are styled via our token system — shadcn's default gray palette is never used.

---

## Prerequisites

- [x] M5.1 complete — shadcn initialized, tokens applied, DM Serif Display loaded, motion.dev installed
- [x] `components.json` configured with correct path aliases
- [x] shadcn CSS variable aliases mapped to our tokens in `globals.css`

All phases (A–E) are implemented. The component library is fully migrated to shadcn/ui with our token system. `pnpm run build` passes with zero errors.
